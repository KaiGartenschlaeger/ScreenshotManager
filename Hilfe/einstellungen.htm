Einstellungen
   
Sie können im Popup Menü im Systrayicon von Screenshot manager unter Einstellungen,
einiges an Benutzerdefinierten Wünschen ändern.

Zwischenablage bei Start leeren
Hierbei leer Screenshot Manager automatisch beim Programmstart die Zwischenablage,
falls sie eine Grafik darin befinden sollte.
Das dient hauptsächlich dazu, das die Automatische Zwischenablage Speicherung
nicht versehntlich beim Programmstart unerwünscht einsetzt.
Um Datenverluste zu vermeiden ist diese Option standardmässig deaktiviert.

Erster Start Fenster anzeigen
Hierbei wird ihnen beim ersten Start eine kurze erklärung zum
aufruf des Hauptfensters eingeblendet.
Sie können dieses Fenster von hieraus jederzeit erneut aktivieren.

Vor beenden fragen
Falls aktivert, wird Screenshot Manager sie erst fragen,
bevor er beendet wird.

Vor speichern fragen
Sicherheitsabfrage, ob die aktuellen Einstellungen übernommen werden sollen.
Ist diese Option deaktiviert, speichert Screenshot Manager die Einstellungen
automatsch beim beenden ab.
