Sprachen
   
Screenshot Manager lässt sich sehr einfach in der Sprache konfigurieren.
Sie können im Handumdrehen neue Sprachen hinzufügen oder vorhandene auswählen.

Sprache verändern
Klicken Sie im Systray mit der rechten Maustaste auf das Screenshot Manager Icon,
um das Popup Menü zu öffnen, und wählen Sie unter Sprache ihre Sprache aus.

Neue Sprache erstellen

Öffnen Sie dazu einfach die Datei Language.ini und tragen Sie dort die neue Sprache ein.
Screenshot Manager wird diese nach einen Neustart dann automatisch zur Auswahl hinzufügen.

Um eine neue Sprache hinzuzufügen,
erstellen Sie zuerst eine neue Gruppe z.B. [Belgisch],
danach kopieren sie einfach von einer vorhandenen Sprache alle nachstehenden Zeilen,
und übersetzen dessen texte.

Alle Zeilen mit g stehen für Fenstergadgets, also die Objekte im Hauptfenster wie z.B. Buttons.
Die Zeilen mit m sind für das Popup Menü im Systray.
d steht dür Dialoge wie z.B. die Frage ob Sie wirklich beenden möchten.
h steht für der aktiven Hilfe in der Statusbar.

Fehler sollte Screenshot Manager beim auswählen der neuen Sprache automatisch erkennen.

Weitergabe der Sprache
Sollten Sie eine Sprache erstellen so geben Sie diese bitte an mich weiter,
ich kann diese dann standardmässig im Programmpacket integrieren,
so das auch andere etwas davon haben.

Schicken Sie diese einfach per Mail an mail@kaispage.de
oder schauen Sie auf http://www.kaispage.de für weiter hinweise nach.
