Text hinzufügen
   
Diese Option erlaubt es ihnen, auf ihren Screenshots von Screenshot Manager
automatisch einen Text hinzufügen zu lassen.
Dabei können Sie die Position des Textes, sowie das Schriftformat frei einstellen.

Wählen Sie die Position auf dem Screenshot aus.
Aktivieren Sie einfach Eigener Text oder Zeit und Datum einfügen.
Wählen Sie die zu verwendende Schriftart mit dem Button Schriftart aus.
