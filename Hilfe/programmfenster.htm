Das Programmfenster
   

Das Programmfenster kann ganz einfach über dem Programmicon im
Systray-Bereich geöffnet werden.
Entweder mit einen Linksklick der Maus auf dem Icon,
oder über dem Popup Menü der rechten Maustaste und Fenster öffnen.

Popup Menü im Systray

Das Programmfenster

Im Programmfenster ist bereits eine aktive Hilfe integriert,
so können sie bequem eine kurze Hilfe zum aktuellen Abschnitt erfahren.
Gehen Sie dazu einfach mit der Maus über einen Objekt,
und in der Statusleiste des Programmfensters wird ein kurzer
Hilfetext erscheinen.

Wichtig ist noch, das sich das Fenster nicht minimiert wenn Sie auf
den minimize Button klicken,
sondern komplett verschwindet.
Sie können das Fenster dann jedoch jederzeit erneut über dem
im Systray befindlichen Icon wieder hervorrufen.
