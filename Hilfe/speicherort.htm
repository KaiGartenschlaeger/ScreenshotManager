Speicherort
   
Hier können Sie festlegen wo Screenshot Manager die Screenshots speichern soll.
Standardmässig ist der export Ordner im Programmverzeichnis aktiviert.
Möchten Sie einen eigenen benutzen so deaktivieren Sie die Checkbox Export Ordner benutzen
und klicken Sie auf dem Verzeichniss Auswahl Button um einen beliebigen Ordner anzugeben.

Der jeweils aktuelle Speicherpath wird ihnen im Textcontrol jederzeit angezeigt.
