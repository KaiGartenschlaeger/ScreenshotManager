Screenshot Taste
   
Diese Option legt fest welche Tastenkombination Screenshot Manager benutzen soll,
um einen Screenshot anzulegen.

Möglich sind alle Kombinationen und Tasten.

Falls Sie eine bestimmte Taste vermissen, so kann diese selbst hinzugefügt werden.
Öffnen Sie dazu die Datei ScreenKeys.ini im Programmordner und fügen Sie folgendes hinzu.

Tastenname zb. Einfügen , Tastencode
