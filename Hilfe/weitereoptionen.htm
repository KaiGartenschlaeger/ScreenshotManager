Weitere Optionen
   
Unter diesen Abschnitt können spezielle Optionen festlegen.

Signalton beim speichern
Hiermit wird ein kurzer Sound abgespielt,
sobald Screenshot Manager einen Screenshot erstellt.
Um einen anderen Sound zu verwenden,
wechseln Sie einfach die Wave Datei SaveSound.wav im Programmordner aus.

Grafiken auf Desktopauflösung testen
Diese Option dient im Zusammenhang mit der Option
Automatisch Zwischenablage speichern dazu,
das jede Grafik in der Zwischenablage erst auf der Größe geprüft wird.
Sollte die Größe nicht der aktuellen Desktopauflösung entsprechen,
so wird diese Grafik einfach ignoriert.

Automatisch Zwischenablage speichern
Diese Option dient dazu, das Screenshot Manager selbständig alle in der
Zwischenablage befindlichen Grafiken sofort abspeichert,
und die Zwischenablage wieder leert.
So kann man z.B. sehr bequem im Internet schnell und einfach Grafiken suchen
und zur späteren benutzung sichern lassen.
